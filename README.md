# UI

## Dependencies

- [NodeJS](http://www.nodejs.org/)
- [Ruby](http://rubyinstaller.org/)

## Tools

- Grunt: `npm install -g grunt-cli`
- Compass: `gem install compass`
- Sass: `gem install sass`

## Run

```
npm install && bower install
grunt serve
```

## Grunt Commands

- run: `grunt serve`
- build: `grunt build`
- build then run: `grunt serve:dist`
